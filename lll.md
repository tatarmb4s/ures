Commit:
Módosítás a git-ben, van szerzője, neve, dátum, leírása

Push:
Az a művelet amikor a gépeden lévő fájlállományt / repo-t feltöltöd a szerverre

Branch:
A git egyik legnagyobb erőssége, hogy lehet több ágat is létrehozni, és a fejlesztés során a fejlesztők külön-külön dolgozhatnak a különböző ágakon, és csak a végén össze lehet őket egyesíteni.

Fetch:
Lekéri a módosításokat a git szervertől, de nem húzza le a gépedre

Pull:
Az amikor a GIT szervertől lehúzod a gépedre a változásokat
